### 自定义事件

1.分类

​	1.标准DOM事件

​	2.自定义事件

2.标准的DOM事件

```bash
# 举例click、input
# 事件名是固定的，事件由浏览器触发
```

3.自定义事件

​	1.绑定事件

```bash
	* 1.事件名
	* 2.事件的回调
	* 3.订阅方:PubSub.subscribe('事件名', 事件的回调函数);
	* 4.订阅方是接收数据的一方
```

​	2.触发事件

* 1.事件名
* 2.提供事件参数对象，等同于原生事件的event对象
* 3.发布方：PubSub.publish('事件名', '提供的数据');
* 4.发布方是提供数据的一方
